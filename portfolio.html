MDVD - Multiple Device Virtual Display

My Dissertation project, implementing the concept of placing a number of Android devices edge-to-edge to form a larger display. In this basic application, the display functions to show an image over the devices.

The devices are interconnected using P2P (Peer-to-Peer) with one acting as a server sharing parts of the image with the other client devices. The relative positions of each device are calculated from a number of pinch gestures between the screens of 2 adjacent devices.

GifMeAMovie

A web application that returns GIFs summarising a given movie. We pre-parsed ~2000 movie scripts found using a python web crawler, ran our Natural Language Processing algorithm on them, and stored the resulting keywords in a database, to be indexed at the user's request.

This was the result of a year long group project I had to complete at university, in which we had to pitch for, design and implement a piece of software for an industry client.

My contributions included constructing parts of the NLP algorithm, and designing/implementing the whole front end, from HTML, CSS and responsiveness oriented Javascript.

I led the team throughout the year, where we used industry tools like Git and Trello, while following the Scrum framework and participating in periodic meetings with the client. At the end of the year we had to present the software at an open day to all the yeargroups clients.
